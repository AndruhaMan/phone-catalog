# Phone catalog
This this a simple, but beautiful react application. I created catalog using a given design. You can use it to search devices, add them to favourites and add thm to the cart.
## Tech stack
  - React
  - TypeScript
  - SCSS
  - React Router
  - Local storage
  - Moibile and tablet adaptation
